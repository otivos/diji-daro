Instructions

Please read the instructions below before uploading.

  1. Ensure that your document is scanned and saved as one PDF file. If you don't have a scanner, you can user your phone's camera to scan the document and convert it to a PDF using a free app like Adobe Scan or CamScanner.
  2. Make sure that you include the cover page/front page of the document which contains the name of the subject, school and year. Our system uses the first page to check the validity of the document.
  3. Make sure the file size does not exceed 3 MB. If the file is too large, you can compress it using a tool like Smallpdf or Adobe Acrobat.
  4. Ensure the subject's name, type, school and year of the document corresponds to what you have selected in the form field. Note: only matching documents will get uploaded to the plattform.
  5. Click the upload button and wait for the process to complete. Depending on the file size and the speed of your internet connection, it may take a few seconds or several minutes. You will receive a confirmation message once the upload is complete.
